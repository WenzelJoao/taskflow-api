    Sobre o projeto

Este projeto consiste no desenvolvimento de uma API backend para gerenciamento de tarefas em equipe, inspirada em ferramentas como Trello.

A aplicação permite que usuários criem e organizem fluxos de trabalho através de quadros (boards), listas e tarefas (cards), facilitando o acompanhamento e a colaboração em atividades.

    Objetivo

O principal objetivo deste projeto é simular um ambiente real de desenvolvimento em equipe, onde múltiplos desenvolvedores trabalham simultaneamente no mesmo código.

Durante o desenvolvimento, são aplicadas práticas comuns do mercado, como:

Uso de Git com estratégia de branches (main, develop, feature/*)
Abertura de Pull Requests (PRs)
Code review entre os membros
Resolução de conflitos de merge
Padronização de commits

    Funcionalidades

Cadastro e autenticação de usuários
Criação e gerenciamento de boards
Organização de listas dentro dos boards
Criação e edição de tarefas (cards)
Comentários em tarefas
Controle de acesso por usuário

    Equipe

Projeto desenvolvido em grupo como parte de atividade prática para simulação de fluxo de trabalho colaborativo em desenvolvimento backend.

    Tecnologias

(Node.js, Express, PostgreSQL, Prisma.)
